DOM (Document Object Model)
 is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects, allowing programming languages to interact with the page.

// dom maths
// document object model
// document is the root element of the html document
// dom is not a part of the html syntax it is a programming interface for html and xml documents
// dom is used to connect web pages to scripts like javascript
// dom is used to add interactivity to a web page

// document object methods
// getElementsByTagName - returns a collection of all elements with the given tag name
// getElementsByClassName - returns a collection of all elements with the given class name
// getElementById - returns the element with the given id

// dom properties
// innerHTML - sets or returns the content of an element as html
// value - sets or returns the value of an element

// dom methods
// getAttribute - returns the value of an attribute
// setAttribute - sets the value of an attribute

// dom events
// onclick - fires when an element is clicked
// onchange - fires when an element is changed
// onmouseover - fires when the mouse is moved over an element
// onmouseout - fires when the mouse is moved away from an element
// onkeydown - fires when a key is pressed
// onkeyup - fires when a key is released
